# Movie Verse (project)

This repository contains the Movie Verse web app (Vite + React + TypeScript).

How to edit locally

1. Clone the repository:

```sh
git clone https://github.com/balkrishan99/movie-verse.git
cd movie-verse
```

2. Install dependencies and start the dev server:

```sh
npm install
npm run dev
```

3. Make changes in your preferred editor, commit, and push.

Deploying with GitHub Pages

This project is set up to publish the production build to the `gh-pages` branch. Use:

```sh
npm run build
# then deploy the contents of `dist/` to the gh-pages branch
```

If you prefer automated deploys, enable a GitHub Action to build and deploy on push to `main`.

Technologies

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
